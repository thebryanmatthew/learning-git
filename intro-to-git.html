Intro to Git

Terms and Definitions

git init
initializes/creates git repository in current directory
status
tells of changes and info about current state of repository
add
prepares a change to be committed
commit
creates a uniquely identified batch of changes
log
shows all commits
push
sends commits in branch to server
fetch
downloads commits in all branch to local
remote add
configure remote repository
remote -v
list all remotes
config --global -e
edits global git config with configured editor
